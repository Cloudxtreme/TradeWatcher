About the tracker...
Following are some EverQuest auctions from the Project 1999 player run classic EverQuest (old school MMORPG) server.

Want to help? Check out the following downloads:
  • eqAuction.windows.zip - unzip and double click (maybe modify your eqAuction.ini file) - so easy Windows users can do it! (zip)
  • eqAuction.linux.7z C client source code for linux machines (7zip)
  • eqAuction.sh Shell Script (Linux/Unix/MAC/BSD)
You can use the above to consistently upload your log file (requires libcurl).Both files require no more setup than pointing to your log file (or directory in the case of the shell script).

New listings will auto-load, don't worry about refreshing!

Not seeing new listings? Contribute your log files using one of the programs listed above or upload directly!
